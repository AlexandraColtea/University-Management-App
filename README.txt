READ ME 
Acesta este un document README pentru o aplicație Java pentru platforma de studiu MySQL.

Aplicația este o platformă care permite utilizatorilor să creeze și să gestioneze grupuri de studiu și resurse.
Aceasta utilizează o bază de date MySQL pentru a stoca informații despre utilizatori, date despre grupuri și materiale de resurse.

Aplicația este construită cu ajutorul limbajului de programare Java. Pentru conectivitatea cu baza de date se foloseste
mysql.jar.

Pentru a rula aplicația, este necesar ca Java și MySQL să fie instalate pe computerul dumneavoastră. 
Aplicația poate fi rulată prin executarea comenzii java -jar [numele fișierului jar] în terminal.

Înainte de a rula aplicația, va trebui să configurați conexiunea la baza de date atasand fișierul extern JAR.
Acest fișier poate fi găsit pe internet.

Vă mulțumim că ați utilizat această aplicație Java pentru platforma de studiu MySQL.
